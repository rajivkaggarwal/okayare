This is the repo for okayare.

It is a project for creating and managing OKRs.

This is a learning project employing:

* Vue
* Serverless
* AWS
  * Lambda
  * API Gateway
* MongoDB Atlas
* GraphQL
* Google Firebase Auth


To Deploy Front-end\
In /vue/okayare:\
npm run build\
pip install awscli      <- One time only to install AWS CLI\
npm run deploy


To Deploy Back-end\
In /serverless\
npm run deploy
